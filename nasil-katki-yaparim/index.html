Nasıl Katkı Yaparım ?

Repo GitHub’da duruyor: 

https://github.com/webBoxio/kod.webbox.io

1. Önce repo’yu fork edin,

2. Sonra kendi branch’inizi oluşturun: 

git checkout -b yeni-post

3. Yaptığınız ekleri / değişklikleri commit edin, 

git commit -am "Yeni post"

4. Branch’e push edin, 

git push origin yeni-post

5. Son olarak Pull Request yapın!

Eğer daha detaylı bilgi almak isterseniz: https://help.github.com/articles/fork-a-repo
