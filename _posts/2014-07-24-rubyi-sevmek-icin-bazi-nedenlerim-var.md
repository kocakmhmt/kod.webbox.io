Şimdi kafamıza göre iki komut üreteceğiz:

```ruby
sence = acaba = { true => 'Evet', false => 'Hayır' }
```

Bunlar `sence` ve `acaba`. Haydi kullanalım:

```ruby
acaba[10 == 50]  # => "Hayır"
sence[10 > 5]    # => "Evet"
```
